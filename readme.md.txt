### Fonctionnalités suplémentaires
Le site est entièrement responsif et peut donc etre utilisé sur téléphone. Sur ce genre d'application web un bon développeur va faire le choix de ne pas utiliser un framework qui peut être lourd et couteux surtout pour ce type d'application qui peut se faire entièrement en vanilla. De ce fait même mon css et entièrement fait à la main. Afin de correspondre au mieux à une véritable application et pour qu'elle soit la moins couteuse en ressources.
